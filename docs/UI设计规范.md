# AI 视频重塑工具 — UI 设计规范

> 设计风格融合：RunwayML（电影级 AI 创意工具）+ Linear.app（深色产品工作台）
> 核心理念：电影感的暗色创意空间 + 专业高效的产品化操作界面

---

## 1. 设计理念

本产品是一个 AI 视频创意工具。用户在其中完成的是「创作」行为，不是「管理」行为。UI 应该让视频内容成为视觉主角，界面本身退为背景框架。

**三个关键词：**
- **沉浸** — 深色环境让视频预览和 AI 生成内容成为视觉焦点
- **高效** — 分镜编辑和批量操作需要清晰的信息层级
- **专业** — 视觉语言传递「这是专业创作工具」的信号

## 6. 动效规范

| 类型 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| 微交互 | 150ms | ease-out | 按钮 hover、切换 |
| 面板展开 | 250ms | ease-in-out | 侧面板、下拉菜单 |
| 页面转场 | 300ms | ease-out | 路由切换 |
| 加载态 | 循环 | linear | spinner、进度条 |

**原则：**
- 不使用弹性动画（过于花哨）
- 不使用超过 400ms 的动画（阻碍效率）
- 进度反馈优先于装饰动画

## 7. 页面级设计指引

### 7.1 登录页

- 全屏深色背景
- 居中登录卡片 (surface-1, 最大 400px 宽)
- 产品名 + 一句话 slogan
- 邮箱 + 密码输入
- 唯一使用品牌渐变的页面（微妙的背景光效）

### 7.2 项目列表页（首页）

- 顶部导航: logo + 用户头像 + 积分余额
- 「新建项目」大按钮 (accent)
- 项目网格: 3列卡片
- 卡片显示: 封面 + 名称 + 状态 + 进度

### 7.3 分镜列表页（项目详情）

- 左侧: 原视频预览 (固定位置)
- 右侧: 分镜卡片列表 (垂直滚动)
- 顶部: 项目名 + 批量生成按钮 + 合并导出按钮
- 底部: 积分余额提示

### 7.4 分镜编辑页

- 上半部: 视频预览（原视频片段 + 生成结果对比）
- 下半部: 编辑面板
  - 左列: 人物信息 + 场景描述 + 对白
  - 右列: Prompt 编辑器 + 素材网格 + 生成参数
- 底部固定: 生成按钮 + 状态提示

### 7.5 合并导出页

- 分镜视频时间线（横向排列）
- 完整视频预览播放器
- 导出按钮 + 进度条
- 下载按钮

## 8. 暗色主题专属规则

1. **永远不要用纯白 (#ffffff)**。最亮的文字色是 `#f4f4f5`。
2. **图片和视频是唯一的亮色来源**。界面本身保持克制。
3. **边框用 1px**，绝不用 2px 以上（除焦点环）。
4. **不使用 box-shadow 做层级**。层级通过表面色阶区分。
5. **强调色（indigo）极度克制使用**：仅用于主按钮、链接、焦点环、选中态。不用作背景色。
6. **hover 状态**：通过边框变亮或背景提亮一级来表达，不用改变文字颜色。
